Personal dot-files managed as a [dot-home](https://github.com/dot-home/_dot-home)
module.

## Bootstrapping

    mkdir -p ~/.home                                    &&
    cd ~/.home                                          &&
    git clone https://github.com/dot-home/_dot-home     &&
    git clone https://github.com/nishantjr/dot-files    &&
    _dot-home/bin/dot-home-setup
